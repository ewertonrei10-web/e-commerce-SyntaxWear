# 👟 SyntaxWear - Tênis e Sneakers Online

Bem-vindo ao repositório do **SyntaxWear**, um E-commerce moderno e elegante especializado em tênis e sneakers. Este projeto foi desenvolvido com foco em performance, responsividade e uma experiência de usuário imersiva.

---

## 🚀 Sobre o Projeto

O SyntaxWear é uma vitrine digital para entusiastas de calçados urbanos, esportivos e modernos. O site apresenta uma interface limpa, com navegação intuitiva e seções bem definidas para facilitar a escolha do modelo ideal.

### ✨ Funcionalidades Principais
- **Design Responsivo:** Adaptado para dispositivos móveis, tablets e desktops.
- **Menu Mobile:** Menu lateral (hambúrguer) funcional para facilitar a navegação em telas menores.
- **Categorização Inteligente:** Seções dedicadas para categorias Masculino, Feminino e Outlet.
- **Seção Hero de Alto Impacto:** Chamadas para ação (CTAs) estratégicas para converter visitantes.
- **Grade de Produtos Dinâmica:** Layout em grid moderno para exibição de coleções em destaque.
- **Newsletter:** Área dedicada para captura de leads e fidelização de clientes.
- **Footer Completo:** Links rápidos, redes sociais e informações institucionais.

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as melhores práticas de desenvolvimento web front-end:

- **HTML5:** Estruturação semântica do conteúdo.
- **CSS3:** Estilização avançada com o uso de:
  - **Variáveis CSS:** Para manutenção simplificada de cores e tipografia.
  - **Flexbox & CSS Grid:** Para layouts robustos e responsivos.
  - **Arquitetura de Componentes:** Estilos organizados por componentes (Header, Footer, Hero, etc.).
- **SVG:** Ícones e logo escaláveis para máxima qualidade visual.

## 📁 Estrutura de Pastas

```text
e-commerce-SyntaxWear/
├── css/                   # Estilos globais e componentes

└── README.md              # Documentação do projeto
```

## 💻 Como Executar o Projeto

Para visualizar o projeto localmente, siga estes passos simples:

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/ewertonrei10-web/e-commerce-SyntaxWear.git
   ```
2. **Navegue até a pasta do projeto:**
   ```bash
   cd e-commerce-SyntaxWear
   ```
3. **Abra o arquivo `index.html` em seu navegador:**
   - Você pode simplesmente clicar duas vezes no arquivo `index.html` ou utilizar uma extensão como **Live Server** no VS Code para uma melhor experiência de desenvolvimento.
